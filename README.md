# Artetxe_Ainhoa_Portfolio
## Master's in Bioinformatics Portfolio | UNIR

¡Bienvenido/a a mi repositorio de proyectos! En este espacio documento mi evolución técnica y científica a lo largo del Máster en Bioinformática de la UNIR. Aquí encontrarás soluciones computacionales aplicadas a retos biológicos, con un enfoque especial en la integración de datos y la reproducibilidad.

## Proyecto Transversal: Enfoque Multidisciplinar

El núcleo de este repositorio es un proyecto transversal diseñado para dar solución a un problema biológico/sanitario central mediante diferentes vías de análisis. Este enfoque permite abordar la complejidad de los datos ómicos desde cuatro perspectivas complementarias:

### 1. Algoritmos e Inteligencia Artificial

- Implementación de modelos predictivos y algoritmos de optimización para la clasificación de muestras o predicción de fenotipos.
- Uso de aprendizaje supervisado y no supervisado para extraer patrones en grandes volúmenes de datos.

### 2. Estadística y R para Ciencias de la Salud

- Validación estadística de hallazgos, análisis de significancia y diseño experimental.
- Tratamiento de datos con rigor bioestadístico para asegurar la robustez de las conclusiones.

### 3. Genética Clínica y de Poblaciones

- Interpretación de variantes genéticas en contextos clínicos.
- Análisis de la estructura poblacional y frecuencias alélicas para entender la base genética de enfermedades.

### 4. Secuenciación y Ómicas de Próxima Generación (NGS)

- Desarrollo de pipelines para el procesamiento de datos crudos (FASTQ/BAM/VCF).
- Análisis de variantes y anotación funcional utilizando herramientas de última generación.

## Otros Proyectos y Scripts
Además del proyecto central, este repositorio contiene:

- Tratamiento de Expresión Génica: Scripts avanzados en R para la normalización, análisis de expresión diferencial (DESeq2/EdgeR) y visualización de datos transcriptómicos (Heatmaps, Volcano plots, etc.).
- Utilidades en Bash/Python: Herramientas de soporte para la automatización de tareas en entornos Linux.

## Stack Tecnológico

- Lenguajes: R, Python, Bash.
- Herramientas Ómicas: Bioconductor, Tidyverse, herramientas de alineamiento y llamada de variantes.
- Entornos: Jupyter Notebooks, RStudio, Terminal Linux.
- Control de Versiones: Git & GitHub.
